# Task 2 - Image Encryption & Decryption Tool (Java)

This project is created for my Cyber Security Internship Task 2.

It is a simple Java program that encrypts and decrypts images using pixel manipulation.

## Description
The program reads an image, changes pixel values using a key to encrypt it, and then reverses the process to decrypt it back to original.

## Features
- Read image from file
- Encrypt image using key
- Decrypt image back
- Save encrypted image as encrypted.png
- Save decrypted image as decrypted.png

## Technologies Used
Java, BufferedImage, ImageIO

## Project Files

## How to Run
javac ImageEncryption.java
